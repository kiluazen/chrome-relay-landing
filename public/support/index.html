Chrome Relay

Support

Chrome Relay is a local developer tool. For help, email kushalsokke@gmail.com with your operating system, Chrome version, extension version, and the output of chrome-relay doctor.

Install checklist

  1. Add Chrome Relay from the Chrome Web Store.
  2. Install the CLI package.
  3. Register the native host.
  4. Open the extension popup and confirm the native host is connected.
pnpm add -g chrome-relay
chrome-relay install
chrome-relay doctor
chrome-relay tabs

Common fixes

  • If the popup says the host is disconnected, run chrome-relay install again and reload the extension.
  • If no tabs appear, confirm Chrome is open in the same user account where the CLI was installed.
  • If a command times out, refresh the target tab and run chrome-relay doctor.
