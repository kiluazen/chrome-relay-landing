Use

Chrome Relay is provided as a local developer tool for connecting Chrome to coding agents you configure. You are responsible for the actions you ask your agent to perform in your browser.

Local agent behavior

Chrome Relay does not decide what your agent should do. It exposes browser-control tools locally. Review your agent's privacy, security, and confirmation behavior before using Chrome Relay on sensitive sites.

Chrome Relay is provided as-is, without warranties of any kind. It may fail, time out, misread a page, or interact with an element differently than expected.
